# Latihanuklsulit
Program Nilai Rata-Rata

Fungsi:
Untuk membantu guru menghitung nilai rata-rata ujian dari sejumlah siswa.

Cara kerja:
Program meminta pengguna memasukkan jumlah siswa, lalu memasukkan nilai masing-masing siswa satu per satu. Semua nilai dijumlahkan, kemudian dibagi dengan jumlah siswa untuk mendapatkan nilai rata-rata yang ditampilkan di layar.

Program Penjumlahan Dua Matriks

Fungsi:
Untuk menjumlahkan dua buah matriks yang dimasukkan oleh pengguna.

Cara kerja:
Program meminta pengguna menentukan ukuran matriks (baris dan kolom), lalu mengisi elemen-elemen dari kedua matriks. Setelah itu, program menambahkan elemen-elemen yang posisinya sama dan menampilkan hasil penjumlahannya dalam bentuk matriks baru.
